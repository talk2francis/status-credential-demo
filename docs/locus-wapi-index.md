## How Wrapped APIs Work

1. Get a Locus API key (`claw_` prefix)
2. Fund your Locus wallet with USDC on Base
3. Call `POST {apiBase}/wrapped/{provider}/{endpoint}` with your API key
4. Locus proxies the request, charges your wallet, returns the response

All calls use the same pattern:

```bash
curl -X POST https://beta-api.paywithlocus.com/api/wrapped/<provider>/<endpoint> \
  -H "Authorization: Bearer YOUR_LOCUS_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ ...parameters... }'
```
